1.Why is Wikipedia a good example of hypertext?

2.Name two reasons why the following path and file name is not optimal: C:\My Documents\File.html
The first reason the file name is not optimal is because the letters are capitlize. The second reason why the path is not optimal is because the file contains space between "My" and "Document".

3.Describe two ways a computer science tree, such as the tree we use to describe the content of an HTML document, is different from a real tree.
The difference between the computer science tree and a real tree is the fact the computer science tree is structured upside down. HTML is the root of the tree and has branches that connect to the elements that have leafs which are the text and attributes. The text and attributes are with in the elements such as body, head and title. The elements are with in the roots unlike a real tree.
